Introduction

  • Web application developed in PHP 5.5, mySql 5.6 & Apache 2.5

    Objectives

  • To help an individual to enrich knowledge in various fields.
  • Allows individual to register, login and play a quiz game.

Overview of Quiz

  • There are various areas of interest where an individual can select one of them at one time and play a game.
  • Each interest field is provided with 10 multiple choice questions, when a individual chooses correct answer the score will be summed where full score will be 10.
  • An individual is provided 60 seconds to solve all 10 questions.
  • If he/she can’t solve within 60 seconds the game is terminated and cannot choose ‘next’ to play.

Used technologies

  • JavaScript & jQuery

    • For client side validations javascript and jQuery are used.
    • A separate js file is created to contains js,jQuery and ajax calls. This also maintains maintainability of code.

  • Ajax

    • The jQuery’s $.ajax() function is extensively used to perform an asynchronous HTTP request with server in client side.

Used technologies cont..

  • gitHub

    • For a version control, log management and code quality review github is used.
    • https://github.com/sgrspkt/quiz-app
